# 🍽️ RestoHub – Restaurant Food Ordering System

RestoHub is a modern and user-friendly restaurant food ordering web application.  
It allows users to browse delicious dishes, view food details, add items to the cart, and place orders easily.  
This project is fully responsive and customizable.

---

## 🚀 Features

### 🔥 User Features
- Browse food categories
- View food images, descriptions & prices
- Add to Cart / Remove from Cart
- Search food items
- Responsive design (mobile + desktop)

### 🛠️ Admin / Owner Side (if included)
- Add / Update / Delete menu items
- Manage orders
- Dashboard (optional)

## ⚙️ Installation & Setup (For Local Development)

### 1️⃣ Clone the repository
```bash
git clone https://github.com/rati-yadav/RestoHub.git

2️⃣ Navigate into the project
cd RestoHub

3️⃣ Create virtual environment
python -m venv env

4️⃣ Activate environment

Windows:

env\Scripts\activate

5️⃣ Install required packages
pip install -r requirements.txt

6️⃣ Run migrations
python manage.py migrate

7️⃣ Start the server
python manage.py runserver

Now open:
👉 http://127.0.0.1:8000/
